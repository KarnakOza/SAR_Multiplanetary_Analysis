## Why this Exists

Lunar geology mapping faces unique challenges: extreme illumination conditions, complex surface textures, and limited ground truth.

- Standaidizing lunar data processing workflows
- Automating feature extraction across multiple data sources
- Providing reproducible machine learning approaches for lithology classificaiton.

**Features**

- Read M3 ENVI/PDS3 formatted data (.HDR + .IMG)
- Extract spectral profiles from pixels
- Compute band ratio images
- Basic atmospheric/photomeric corrections
